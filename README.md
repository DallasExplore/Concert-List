# 🎵 Concert List

A simple web project that shows a list of concerts in a clean format.

## Features
- Displays concert information
- Simple and beginner-friendly layout
- Easy to expand with styling and JavaScript

## Project Structure
- `index.html` — main webpage
- `README.md` — project documentation

## Tech Stack
- HTML

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
